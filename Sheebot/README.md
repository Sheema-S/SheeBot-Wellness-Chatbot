# SheeBot AI – Wellness Analytics System

SheeBot is an AI-powered wellness assistant that combines conversational AI, behavioural tracking, and real-time dashboards to support mental health and lifestyle monitoring.

## Features
- 🤖 AI-powered chatbot using LLaMA3 and GPT
- 📊 Interactive dashboards (Chart.js)
- 💧 Tracks water intake, sleep, and mood
- 🧠 Context-aware memory system
- 📈 Weekly summaries and insights

## Tech Stack
- Python (Flask)
- JavaScript (Chart.js)
- Ollama (LLaMA3)
- JSON-based local storage

## How to Run

1. Clone the repository
2. Install dependencies
3. Start the AI model (Ollama)
Make sure Ollama is installed and running locally.
In Terminal 1:ollama run llama3
4. Run the Flask application
In Terminal 2:python app.py
5. Open in browser
http://127.0.0.1:5000
